# Neo Stack Pack

This is an extension pack used and maintained for developers at https://www.neofinancial.com/

Included Extensions:

- Editorconfig
- Trailing spaces
- Code spell checker
- Color picker
- Docker
- DotENV
- ESLint
- Git Blame
- Git History
- Github Markdown Preview
- Github Pull Requests
- gitignore
- GraphQL for VSCode
- Image preview
- Prettier
- Quit Control for VSCode
- Rainbow Brackets
- SVG Viewer
- Terraform
- Todo Tree
- Jest Runner
